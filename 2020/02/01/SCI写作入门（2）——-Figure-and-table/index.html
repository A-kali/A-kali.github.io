文字、图片和表格的合理使用

图表可以用来表示

  1. 结果：得出的数据、统计图、实验对比等
  2. 方法：流程图、示意图

图表的分类：

  • 投稿时：一般要求，在尺寸、分辨率、色彩等方面有一定要求，能让审稿人看清就行；
  • 发表时：对图片质量要求高，包括图片文字、数字、字号大小、线条粗细等都有具体的要求

文字表述：

  1. 描述定量化的结果
  2. 用来描述定性数据之间的关系（e.g. 同比上升/下降）

图片表述：

  1. 用来展示数据组之间的趋势
  2. 描述指标随时间改变的情况
  3. 定量化因素之间的复杂关系
  4. 实验方法介绍（流程图）
  5. 复杂概念介绍
  6. 可视化数据结果（e.g. 细胞图片）

表格表述：

  1. 展示大量数值型数据
  2. 不同项目之间的细节比较
  3. 复杂定性结果之间比较

图片

常见图片类型

照片、线型图、柱状图、流程图、示意图等

图片制作过程注意事项

  1. 图标：一个图片对应一个图标。比如说图标a只能对应一个大图中的一个小图。
  2. 标注：箭头、星号；
  3. 缩写：图片标识的时候可以使用缩写，但一定要定义；
  4. 颜色：根据杂志要求选择色彩或灰度模式。

表格

表格的要素

  1. 数值；
  2. 列标题和列；
  3. 行标题和行；
  4. 注释和说明。

论文表格的一般要求

  • 单位：一般情况下，单位标识在行标题的后面；
  • 对比：通常使用横向比较而不是纵向比较；
  • 顺序：时间顺序、大小顺序、字母顺序；
  • 空白数据：可以使用”-“表示，或者相应缩写（e.g. ND, Not Done)。

表格中的线条

  1. 在列标题上下需要一条线条；
  2. 最后一行数据下面需要一条线条；
  3. 列的副标题下面需要线条；
  4. 除了以上三种情况以外，其它的都不能使用线条！

表格中的注释

表格中出现多个注释的时候，可以使用*、**、#等

表格中的数值

  • 如果有百分比数值最好加上百分比数值；
  • 如果样本数很多，最好在列标题中标注样本总数；
  • 如果某一列中所有数据相同，需要在注释中说明指出；
  • 对齐方式：小数点对齐、左圆括号对齐；
  • 多次检查。表格中数据多，容易出错。
