比赛链接：RSNA Intracranial Hemorrhage Detection

经过闭关一个月补习机器学习和计算机视觉之后，某凯再度重出江湖。世事变迁，沧海桑田。Kaggle平台亦是发生了翻天覆地的变化，更新了整整！……1个比赛！让我们看看是哪场比赛这么幸运得我恩宠。

首先映入眼帘的是……一堆脑阔（真 · 头像），看起来又又又又又又双双叒叕是个医学图像类比赛。翻译标题，RSNA估计是某个学会的名字，Intracranial Hemorrhage 则是“颅内的番茄汁（误）”“颅内出血”，最后一个词则是大家熟知的“检测”；副标题为“确定急性颅内出血及其亚型”。从标题可以初步猜测出这是个出血区域的检测任务（并不）。

Description

Evalution

首先看下提交文件的格式：

每张图片有如上 Id 字段下的六个子类型，前五种分别对应上图中的五种出血类型，最后的 any 类型则代表着存在任意一种子类型的可能性；Label 字段则表示该子类型存在的概率。由此可以看出这是个回归问题，但和常规图片分类的思想差别不大。

评估指标为加权多标签对数损失，即对每个id的每个子类型的对数损失加权平均。然后取所有id损失的平均值作为最终分数。
